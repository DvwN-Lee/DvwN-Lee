## 1. Executive Summary

### 핵심 수치 (2026년 3월 기준)

| 지표 | 수치 |
|------|------|
| 잡코리아 LLM 채용공고 | **699건** |
| 잡코리아 AI Agent 채용공고 | **154~262건** |
| 잡코리아 RAG 채용공고 | **321건** |
| 링크드인 코리아 AI Engineer 공고 | **136건** |
| RAG 기술 수요 증가율 (전년비) | **+2,047%** |
| AI 역량 채용 고려 기업 비율 | **69.2%** |
| 경력직 선호 기업 비율 | **51%** |
| 한국 AI 전문인력 추정 규모 | **약 5.7만 명 (공급 부족)** |
| 한국 AI 시장 규모 (2025) | **3조 4,385억 원 (+12.1%)** |

### 3줄 요약

1. **수요 폭증 + 공급 부족**: AI Agent/LLM 직군은 최고 성장 직군이나 전문인력이 부족해 임금 프리미엄과 채용 조건 완화가 동시에 진행 중
2. **경력 중심 시장**: 전체 채용의 51%가 경력 우선, AI Agent 직군은 이보다 더 강한 경력 편중(4~7년차 49.7%)
3. **기술 전환 중**: 단순 도구 사용자(ChatGPT -49%) 수요는 감소하고, 아키텍처 이해 기반 엔지니어(RAG +2,047%, MCP 신규 등장) 수요 폭증

|------|------------|

### 고용 형태 분포

- **경력직 선호**: 51% 기업이 경력 최우선, AI 직군 경력 지원자 +197% 증가
- **집중 채용 연차**: 4~7년차 49.7% / 1~3년차 19.6% / 신입 12.4%
- **신입 환경**: 신입 채용 공고 전년비 -5%, 포트폴리오 없이 합격이 매우 어려운 구조로 고착화
- **정규직 vs 계약직**: AI 분야는 전반적 정규직 감소 추세의 예외 (수요 유지·증가)

### 채용 집중 시기

- 상반기 공고 집중: **2~4월**
- 하반기 공고 집중: **8~10월**
